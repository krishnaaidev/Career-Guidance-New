# Pixel2Predict - Career Guidance System

AI-powered career recommendation system using machine learning.

## Features
- 50+ job roles with detailed skill requirements
- AI-powered skill matching
- Skill gap analysis
- Export recommendations (CSV, JSON, Print)
- Responsive dark theme design
